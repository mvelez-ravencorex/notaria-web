# Agentes - Documentación del Proyecto

Esta carpeta contiene documentación y aprendizajes para el equipo de agentes IA que trabajan en este proyecto.

## Estructura

```
.agents/
├── README.md           # Este archivo
├── context.md          # Contexto del proyecto
├── learnings.md        # Aprendizajes acumulados
├── decisions.md        # Decisiones de diseño
└── patterns.md         # Patrones y convenciones
```

## Propósito

- Mantener contexto persistente entre sesiones
- Documentar decisiones técnicas y de diseño
- Registrar aprendizajes y mejores prácticas
- Facilitar la colaboración entre agentes
